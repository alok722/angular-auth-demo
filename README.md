> this repository contains quick demo of angular auth guard.

## 🛠 usage

### setting up back-end
```sh
$ git clone https://github.com/alok722/angular-auth-demo.git

$ cd backend

$ npm i

$ npm run start
```

### setting up front-end
```sh
$ git clone https://github.com/alok722/angular-auth-demo.git

$ cd frontend

$ npm i

$ npm run start
```
